# Paris Trip

Личный сайт-гид по поездке в Париж.

Публичная версия:
- [https://lekmaby.github.io/paris_trip/](https://lekmaby.github.io/paris_trip/)

## Что внутри

- подробный маршрут;
- краткая версия по дням;
- тёмная и светлая темы;
- мобильная навигация по дням;
- автоматическая публикация в GitHub Pages.

## Структура

- `paris_trip_site/` — статический сайт;
- `.github/workflows/deploy-paris-site.yml` — публикация в GitHub Pages.

## Публикация

Сайт публикуется через GitHub Actions из папки `paris_trip_site`.
